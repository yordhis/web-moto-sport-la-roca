Este directorio contiene componentes React usados en la página.

- `Header.jsx`: cabecera y navegación
- `Hero.jsx`: sección hero con imagen de fondo
- `About.jsx`: sección quienes somos con iconos
- `Stores.jsx`: sección tiendas con imagen
- `Brands.jsx`: sección marcas y CTA
- `Footer.jsx`: pie de página

Recomendación: reemplazar textos e imágenes por contenido real y extraer constantes a un archivo de configuración si crece el proyecto.
